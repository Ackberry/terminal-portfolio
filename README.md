# Deep's Terminal Portfolio

A Termux-styled interactive terminal portfolio website built with TypeScript and Vite.

## Features

- **Terminal-style interface** - Looks and feels like Termux
- **Interactive commands** - Navigate using familiar shell commands
- **Virtual filesystem** - Browse through sections like directories
- **Easter eggs** - Try `sudo rm -rf /`, `vim`, `matrix`, and more!

## Commands

### Navigation
- `ls` - List directory contents
- `cd <dir>` - Change directory
- `pwd` - Print working directory
- `cat <file>` - Display file contents

### Information
- `help` - Show available commands
- `whoami` - Display user info
- `neofetch` - Show system info with ASCII art
- `open <github|linkedin|email>` - Open links

### Utilities
- `clear` - Clear screen
- `history` - Show command history
- `echo <text>` - Echo text
- `date` - Show current date
- `uptime` - Show session uptime

### Fun
- `sudo` - Try it!
- `vim` - Classic
- `matrix` - Matrix rain effect
- `hack` - Hacking simulation
- `cowsay <message>` - Have a cow say something

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

The built files in `dist/` can be deployed to any static hosting:
- Vercel
- Netlify
- GitHub Pages
- Cloudflare Pages

## Tech Stack

- TypeScript
- Vite
- Vanilla CSS (Termux theme)
